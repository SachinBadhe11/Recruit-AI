# Metrics & Roadmap - Recruit-AI

## Success Metrics (KPIs)
How do we measure if Recruit-AI is solving the problem?

1.  **Time-to-Interview (North Star):**
    - *Goal:* Reduce from 5 days to <24 hours.
    - *Why:* Speed is the biggest competitive advantage in hiring.
2.  **Screening Efficiency:**
    - *Goal:* Reduce recruiter time spent on screening by 90%.
    - *Measurement:* Hours spent per hire (Manual vs. AI).
3.  **Candidate Satisfaction (NPS):**
    - *Goal:* > 70 NPS.
    - *Why:* Candidates hate "black holes". Instant feedback/scheduling improves their experience.

## Product Roadmap

### Phase 1: The MVP (Current)
- [x] Resume Parsing (Text)
- [x] JD Analysis
- [x] Scoring & Summarization (0-100)
- [x] Basic UI (Upload & View Results)

### Phase 2: Automation (Month 2-3)
- [ ] **Calendar Integration:** Connect Google/Outlook Calendar to auto-schedule interviews for candidates scoring >85.
- [ ] **Email Agents:** Auto-send rejection emails (personalized) and interview invites.
- [ ] **Multi-Resume Upload:** Bulk upload 50 resumes at once.

### Phase 3: Intelligence (Month 4-6)
- [ ] **Voice Screening:** AI calls the candidate for a 5-min pre-screen.
- [ ] **Bias Detection:** Advanced analytics to ensure fair hiring practices.
- [ ] **ATS Integration:** Two-way sync with Greenhouse/Lever.
